# Assets Directory

## Required Assets

This directory should contain the following image files:

1. **icon.png** - 1024x1024px app icon
2. **splash.png** - 1242x2436px splash screen (recommended size)
3. **adaptive-icon.png** - 1024x1024px Android adaptive icon
4. **favicon.png** - 48x48px web favicon

## Creating Assets

You can create these assets using:

1. **Online Tools:**
   - [AppIcon.co](https://www.appicon.co/) - Generate all app icons
   - [IconKitchen](https://icon.kitchen/) - Android adaptive icons
   - [Favicon Generator](https://realfavicongenerator.net/) - Web favicons

2. **Design Tools:**
   - Figma, Sketch, or Adobe XD
   - Export at the required sizes

3. **Quick Placeholders:**
   For development, you can use simple colored squares. The app will work with placeholder images, but you should replace them with proper designs before production.

## Current Status

Placeholder files have been created. Replace them with actual images before building for production.
